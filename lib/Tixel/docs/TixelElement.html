Class: TixelElement

TixelElement

new TixelElement(config, config)

Tixel provides a jQuery-like interface for dynamic manipulation of an image, represented as an array of pixels. Initialize either with size array or with configuration object.
Parameters:
Name Type Description
config array Size of matrix as [width, height], default: [10,10]
config object Configuration object, all values are optional, however a size should be passed
Properties
Name Type Description
size array Size of matrix, default: [10,10]
size[0] number Element width
size[1] number Element height
origin array Position of origin of shapes on matrix, default: [0,0]
origin[0] number Origin x value
origin[1] number Origin y value
defaultColor string Default color for background, default: '000000'
colorMap array Array of color which is mapped on values for output generation
Properties:
Name Type Description
size array Size of output (pixel matrix)
size[0] number Width of output in pixels
size[1] number Height of output in pixels
origin array Origin coordinate of canvas, can be used to position child TixelElement
origin[0] number Origin x coordinate
origin[1] number Origin y coordinate
defaultColor string Default/background color
values ndarray Value matrix is used to calculate output based on colorMap (Array based)
opacity ndarray Opacity of element, can be used for child TixelElement (Float64Array based)
output ndarray Rendered matrix of colors, output of TixelElement (Array based)
colorMap array List of colors which is used map values
childElements array Child TixelElements which are rendered on given pixel element
events EventEmitter Emit events
frame number Frame index, gets incremented on each render
Source:
  • TixelElement.js, line 61
Example
     var tixel = new TixelElement([10,10]); // initialize with size array
     var tixel = new TixelElement({         // initialize with object
         size: [10,10],
         defaultColor: 'FF0000',
         colorMap: ['00FF00', '00FF00']
     });
     
     tixel.selectLine([0,0], [10,10]).setSelectedPixels('00FF00'); // draw a line directly
     var output = tixel.render();
     
     tixel.selectLine([0,0], [10,10]).setSelectedValues(1); // draw a line by using values which are converted to colors
     var output = tixel.render();
