Class: TixelSelection

TixelSelection

new TixelSelection(config, config)

TixelSelection provides an API for convinient selection of cells in 2D matrix. This is is used by pixel to easily select multiple pixels, e.g. in a box shape, optionally change that shape, e.g. by make selection grow by one pixel, to apply a command on all that pixels in Tixel, e.g. color('FFFFFF').
